# PDQ Employee API Technical Exercise
***
[Find live version of exercise here](https://pdq-assignment.herokuapp.com)

This exercise was done using Nodejs, Express, EJS, Node Fetch, and Node HTML Parser.

### Tasks 
- [x] Fetch Employee Name and Favorite Beer
- [x] Reach out to 'About Us' page, parse HTML, find image sources
- [x] Render name, favorite beer, and image to front end (Used server side rendered pages with EJS)
- [x] Update all front end clients when API call button is clicked (Used SocketIO to listen on updated and update all client side pages open)
- [x] Error handling
